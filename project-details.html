Image Recognition Model

Description:
A CNN-based model built with MobileNetV2 to classify images from the CIFAR-10 dataset. The model leverages transfer learning and advanced data augmentation to achieve high accuracy on real-world image classification tasks.

Technologies & Tools:
Python, TensorFlow, Keras, MobileNetV2, Computer Vision, Data Augmentation

Result:
Achieved over 98% accuracy on the CIFAR-10 test set.
